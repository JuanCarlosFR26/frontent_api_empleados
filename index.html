Documentación de la API de Empresa

Nota: Esta API no tiene persistencia de datos. Los cambios se perderán al reiniciar el servidor.

GET /

Obtiene una lista de toda la información de la API

https://api-empleados-empresa-v9t9.vercel.app

GET /departamentos

Obtiene una lista de todos los departamentos.

https://api-empleados-empresa-v9t9.vercel.app/departamentos

GET /empleados

Obtiene una lista de todos los empleados de todos los departamentos.

https://api-empleados-empresa-v9t9.vercel.app/empleados

POST /empleados/:id_departamento

Añade un nuevo empleado al departamento especificado. Debe incluir un objeto empleado en el cuerpo de la solicitud.

https://api-empleados-empresa-v9t9.vercel.app/:id_departamento -H "Content-Type: application/json" -d '{"nombre_empleado": "Nombre", "apellido_empleado": "Apellido", "email_empleado": "email@example.com", "genero_empleado": "Hombre", "puesto_empleado": "desarrollador", "salario_anual_empleado": >= 24000, "seguro_empleado": true o false, "codigo_acceso": "a1a1-b2b2"}'

PUT /empleados/:id_departamento/:id_empleado

Actualiza la información de un empleado específico.

https://api-empleados-empresa-v9t9.vercel.app/:id_departamento/:id_empleado -H "Content-Type: application/json" -d '{"nombre_empleado": "Nombre", "apellido_empleado": "Apellido", "email_empleado": "email@example.com", "genero_empleado": "Hombre", "puesto_empleado": "desarrollador", "salario_anual_empleado": >= 24000, "seguro_empleado": true o false, "codigo_acceso": "a1a1-b2b2"}'

DELETE /empleados/:id_departamento/:id_empleado

Elimina un empleado específico de un departamento.

https://api-empleados-empresa-v9t9.vercel.app/:id_departamento/:id_empleado
Nota: Puedes probar a crear una base de datos para guardar la API, y así tener persistencia de datos.
